# YFYD — Youth For Youth Development

Premium, modern, accessible website built with plain HTML, CSS, and vanilla JavaScript. No external dependencies.

## Structure
- `index.html` — Single-page site with sections: Home, About Event, About Us, Sponsors, Vision, Contact
- `assets/css/style.css` — Responsive, premium UI styles with CSS variables
- `assets/js/main.js` — Navigation, smooth scroll, scroll-reveal animations, contact form validation

## Run Locally
Open `index.html` directly in your browser.

```bash
# On Linux (bash), from the project folder
xdg-open index.html
```

## Accessibility
- Skip-to-content link
- Keyboard-friendly focus states
- Reduced motion support
- Semantic landmarks and labels

## Extending
- Replace sponsor placeholders with logos (local images)
- Add event details or new sections following the existing CSS utility patterns
- No frameworks or CDNs used; safe to host statically anywhere.
